1) Boxing: The process of converting/boxing the primitive data type to
           it's corresponding wrapper class. For example: int -> Integer.
           This can be done in two ways either by Auto Boxing or by Manual
           Boxing.

2) Unboxing: The process of converting/unboxing the wrapper class into it's
             corresponding primitive data type. For Example Double -> double.
             This can be done in two ways, either by Auto Unboxing or Manual
             Unboxing.
