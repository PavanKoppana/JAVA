Method: methods in java are generally used to specify a block of code that
        performs a specific task, and the cool part is, this method can
        be re-used n number of times.

Types: Typically there are four types of method:

1) method with no return type and no input arguments.
2) method with no return type and has input arguments.
3) method with a return type and no input arguments.
4) method with a return type and input arguments.
